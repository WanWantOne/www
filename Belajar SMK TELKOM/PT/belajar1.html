3.Penggunaan teks formating HTML

a.membuat teks menjadi tebal menggunakan tag b dan strong

Menggunakan teks tebal pakai tag b, ini contoh penggunaan teks teba menggunakan tag strong

b. membuat teks menjadi miring menggunakan tag i dan em

Menggunakan teks miring pakai tag i, ini contoh penggunaan teks miring menggunakan tag em

c. membuat teks garis bawah menggunakan tag u dan ins

Menggunakan teks garis bawah pakai tag u, dan ini contoh penggunaan teks garis bawah menggunakan tag ins

d. Membuat teks tercoret menggunakan tag del dan s

Menggunakan teks tercoret pakai tag del, dan ini contoh penggunaan teks tercoret menggunakan tag s

e. Membuat teks marker menggunakan tag mark

Marker biasanya digunakan untuk penanda teks itu penting, kalau di tulisan biasa pakai stabilo

f. Membuat pangkat diatas dan pangkat dibawah menggunakan tag sup dan sub

Untuk membuat pangkat diatasdan untuk membuat pangkat dibawah

Rumus luas persegi adalah S 2, di mana S adalah sisi dari persegi. Lalu 0 2 adalah rumus kimia dari oksigen

g. Membuat tanda kutip menggunakan tag q

Ini adalah contoh penggunaan tag q quote tanda kutip diatas

h. Menggabungkan format teks

Penggabungan format teks bisa dilakukan dengan menuliskan tag-tag yang akan dipakai. Misalkan tebal dan garis bawah, maka kira tinggal pakai tag b dan u
